## CorrelationDistance

```
CorrelationDistance(u, v)
```
> returns the correlation distance between `u` and `v`.
  
### Examples

```
>> CorrelationDistance({a, b}, {c, d})
1+(-a*c+b*c+a*d-b*d)/(2*Sqrt(Abs(a+1/2*(-a-b))^2+Abs(1/2*(-a-b)+b)^2)*Sqrt(Abs(c+1/2*(-c-d))^2+Abs(1/2*(-c-d)+d)^2))
```
