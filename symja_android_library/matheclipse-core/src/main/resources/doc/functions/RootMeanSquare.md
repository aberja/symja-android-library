## RootMeanSquare

```
RootMeanSquare(list)
```

> calculate the root mean square
 
See
* [Wikipedia - Root mean square](https://en.wikipedia.org/wiki/Root_mean_square)

### Examples

```
>> RootMeanSquare({x,y,z})
Sqrt(x^2+y^2+z^2)/Sqrt(3)
 
>> RootMeanSquare({42,43})
Sqrt(3613/2)
```
