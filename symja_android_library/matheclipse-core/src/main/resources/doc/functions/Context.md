## Context 

```
Context(symbol)
```

> yields the name of the context where `symbol` is defined in.


```
Context()
```

> return the current context.

### Examples

``` 
>> $ContextPath
{System`,Global`}

>> Context(a)
Global`

>> Context(Sin)
System`
```
