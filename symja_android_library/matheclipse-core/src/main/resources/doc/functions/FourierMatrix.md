## FourierMatrix

``` 
FourierMatrix(n)
```

> gives a fourier matrix with the dimension `n`.

### Examples

```
>> FourierMatrix(4)
{{1/2,1/2,1/2,1/2}, 
 {1/2,I*1/2,-1/2,-I*1/2},  
 {1/2,-1/2,1/2,-1/2}, 
 {1/2,-I*1/2,-1/2,I*1/2}}
```
