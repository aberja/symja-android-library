## PolyLog

```
PolyLog(s, z)
```

> returns the polylogarithm function.

See:
* [Wikipedia - Polylogarithm](https://en.wikipedia.org/wiki/Polylogarithm) 
 
### Examples

```   
>> PolyLog(-3,f(x)) 
(f(x)+4*f(x)^2+f(x)^3)/(1-f(x))^4
```
