## SixJSymbol

```
SixJSymbol({j1,j2,j3},{j4,j5,j6})
```

> get the 6-j symbol coefficients.

See:  
* [Wikipedia - 6-j symbol](https://en.wikipedia.org/wiki/6-j_symbol)   

### Examples

```  
>> SixJSymbol({1, 2, 3}, {1, 2, 3})
1/105
```
