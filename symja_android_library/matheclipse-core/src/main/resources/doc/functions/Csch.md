## Csch

```
Csch(z)
```

> returns the hyperbolic cosecant of `z`.
  
See
* [Wikipedia - Hyperbolic function](https://en.wikipedia.org/wiki/Hyperbolic_function)

### Examples
```
>> Csch(0)  
ComplexInfinity
```
