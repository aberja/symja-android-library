## BitOr

```
BitOr(int1, int2, int3, ...)
```

> returns the bitwise `OR` of the integer numbers `int1, int2, int3, ...`.
 
### Examples

```
>> Table(BitOr(n,3), {n,-10,10})
{-9,-9,-5,-5,-5,-5,-1,-1,-1,-1,3,3,3,3,7,7,7,7,11,11,11}
```
