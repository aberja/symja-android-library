## ModularInverse

```
ModularInverse(k, n)
```

> returns the modular inverse `k^(-1) mod n`.
 
See
* [Wikipedia - Modular multiplicative inverse](https://en.wikipedia.org/wiki/Modular_multiplicative_inverse)

### Examples

```
>> ModularInverse(3, 11)
4
```
