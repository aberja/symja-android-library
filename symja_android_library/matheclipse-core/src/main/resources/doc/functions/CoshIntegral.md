## CoshIntegral

```
CoshIntegral(expr)
```

> returns the hyperbolic cosine integral of `expr`.
  
See
* [Wikipedia - Trigonometric integral](https://en.wikipedia.org/wiki/Trigonometric_integral)

### Examples

```
>> CoshIntegral(0)
-Infinity
```
