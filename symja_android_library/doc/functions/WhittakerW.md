## WhittakerW

```
WhittakerW(a, b, z)
```

> returns the Whittaker function `W_a,b(z)`.

See
* [Wikipedia - Whittaker function](https://en.wikipedia.org/wiki/Whittaker_function)
 
### Examples

```
>> N(WhittakerW(2, 1/2, 2*2*I)) 
-0.6160300298511752+I*17.877933521588048
```
