## HurwitzLerchPhi

```
HurwitzLerchPhi(z, s, a)
```

> returns the Hurwitz-Lerch transcendent function.

See: 
* [Wikipedia - Lerch zeta function](https://en.wikipedia.org/wiki/Lerch_zeta_function) 

### Examples

```
>> HurwitzLerchPhi(-1,-1,1) 
1/4
```
