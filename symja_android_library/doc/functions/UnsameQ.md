## UnsameQ

```
UnsameQ(x, y)

x=!=y
```

> returns `True` if `x` and `y` are not structurally identical.

See
* [Wikipedia - Computer algebra - Equality](https://en.wikipedia.org/wiki/Computer_algebra#Equality)

### Examples

Any object is the same as itself:

```
>> a=!=a
False

>> 1=!=1.0
True
```
