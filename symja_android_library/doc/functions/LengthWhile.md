## LengthWhile

```
LengthWhile({e1, e2, ...}, head)
```

> returns the number of elements `ei` at the start of list for which `head(ei)` returns `True`.

### Examples


```
>> LengthWhile({1, 2, 3, 10, 5, 8, 42, 11}, # < 10 &)
3
```
