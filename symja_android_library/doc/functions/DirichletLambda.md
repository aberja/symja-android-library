## DirichletLambda

```
DirichletLambda(z)
```

> returns the Dirichlet lambda function.
  

### Examples

```
>> Table(DirichletLambda(x), {x, -4, 4})
{0,-7/120,0,1/12,0,ComplexInfinity,Pi^2/8,7/8*Zeta(3),Pi^4/96}
```
