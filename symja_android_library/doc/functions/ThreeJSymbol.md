## ThreeJSymbol

```
ThreeJSymbol({j1,m1},{j2,m2},{j3,m3})
```

> get the 3-j symbol coefficients.

See:  
* [Wikipedia - 3-j symbol](https://en.wikipedia.org/wiki/3-j_symbol)   

### Examples

```  
>> ThreeJSymbol({3/2, -3/2}, {3/2, 3/2}, {1, 0}) 
Sqrt(3/5)/2
```
