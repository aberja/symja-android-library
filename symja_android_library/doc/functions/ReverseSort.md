## ReverseSort

```
ReverseSort(list)
```

> sorts `list` (or the leaves of any other expression) according to reversed canonical ordering.

```
ReverseSort(list, p) 
```

> sorts in reversed order using `p` to determine the order of two elements.
 
### Examples

```
>> ReverseSort({4, 1.0, a, 3+I})
{a,4,3+I,1.0}
```
