## QuantityUnit

```
QuantityUnit(quantity)
```

> return the unit of the `quantity`

See 
* [Wikipedia - International System of Units](https://en.wikipedia.org/wiki/International_System_of_Units)

### Examples
 
```
>> QuantityUnit(Quantity(42, "Kilograms")) 
Kilograms
```
