## Area

```
Area(geometric-form)
```

> returns the area of the `geometric-form`.
  

See:
* [Wikipedia - Area](https://en.wikipedia.org/wiki/Area)
 

### Examples

```
>> Area(Disk({1,2}))
Pi
```
