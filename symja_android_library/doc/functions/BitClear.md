## BitClear

```
BitClear(i, n)
```

> clears the `n`th bit in the integer `i`. 
 
### Examples

``` 
>> IntegerDigits(36,2)
{1,0,0,1,0,0}

>> BitClear(36,2) 
32

>> IntegerDigits(32,2) 
{1,0,0,0,0,0} 

>>BitClear(36,-4)
BitClear(36,-4)
```
