## LerchPhi

```
LerchPhi(z, s, a)
```

> returns the Lerch transcendent function.

See: 
* [Wikipedia - Lerch transcendent](https://en.wikipedia.org/wiki/Lerch_transcendent) 

### Examples

```
>> LerchPhi(-1,1,0)", //
-Log(2)
```
