## GoldenAngle

```
GoldenAngle
```

> is the golden angle `Pi*(3-Sqrt(5))`.

See
* [Wikipedia - Golden angle](https://en.wikipedia.org/wiki/Golden_angle) 
 
### Examples

``` 
>> N(GoldenAngle) 
2.39996
```
